## OHPL 1.4.1

CRAN release: 2024-07-20

### Improvements

- Reformatted roxygen2 documentation to use Markdown syntax and further
  align with the tidyverse style guide
  ([\#6](https://github.com/nanxstats/OHPL/issues/6)).
- Updated `inst/CITATION` to use
  [`bibentry()`](https://rdrr.io/r/utils/bibentry.html) to replace
  [`citEntry()`](https://rdrr.io/r/utils/citEntry.html)
  ([\#5](https://github.com/nanxstats/OHPL/issues/5)).
- Simplified the package help topic by using the special sentinel
  `"_PACKAGE"` ([\#4](https://github.com/nanxstats/OHPL/issues/4)).
