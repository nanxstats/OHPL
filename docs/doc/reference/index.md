# Package index

## Fit OHPL models

Fit OHPL models.

- [`OHPL()`](https://ohpl.io/doc/reference/OHPL.md) : Ordered
  Homogeneity Pursuit Lasso
- [`FOP()`](https://ohpl.io/doc/reference/FOP.md) : Fisher optimal
  partition
- [`proto()`](https://ohpl.io/doc/reference/proto.md) : Extract the
  prototype from each variable group
- [`dlc()`](https://ohpl.io/doc/reference/dlc.md) : Compute D, L, and C
  in the Fisher optimal partitions algorithm
- [`OHPL-package`](https://ohpl.io/doc/reference/OHPL-package.md) :
  OHPL: Ordered Homogeneity Pursuit Lasso for Group Variable Selection

## Evaluate OHPL models

Cross-validation, prediction, performance evaluation, and generation of
simulated data.

- [`cv.OHPL()`](https://ohpl.io/doc/reference/cv.OHPL.md) :
  Cross-validation for Ordered Homogeneity Pursuit Lasso
- [`predict(`*`<OHPL>`*`)`](https://ohpl.io/doc/reference/predict.OHPL.md)
  : Make predictions based on the fitted OHPL model
- [`OHPL.RMSEP()`](https://ohpl.io/doc/reference/OHPL.RMSEP.md) :
  Compute RMSEP, MAE, and Q2 for a test set
- [`OHPL.sim()`](https://ohpl.io/doc/reference/OHPL.sim.md) : Generate
  simulation data for benchmarking sparse regressions (Gaussian
  response)

## Datasets

Real-world spectroscopic datasets used in the paper.

- [`wheat`](https://ohpl.io/doc/reference/wheat.md) : The wheat dataset
